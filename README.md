# GoToCare Caregiver Portal

**Modern mobile-first marketplace PWA** for caregivers to browse care requests, manage availability, and track earnings.

## Features

- 🔐 **Authentication**: Login with email/password
- 📱 **5-Tab Dashboard**: Home, Schedule, Requests, Earnings, Profile
- 💼 **Care Request Cards**: Uber-style matching with scores, pay rates, distance
- 📊 **Earnings Tracking**: Daily/weekly breakdown with analytics
- ⏰ **Availability Toggle**: Set when you're available for requests
- 🎨 **Landing Page Theme**: Unified purple/blue color system
- 📲 **PWA Ready**: Add to home screen for app-like experience

## Deploy to Cloudflare Pages

```bash
npm run build
# Then deploy using Cloudflare Pages UI
```
